# 📊 HR Analytics | Employee Presence Insights Dashboard

An interactive **Power BI dashboard** designed to analyze employee attendance, work-from-home trends, and leave utilization. This project transforms raw attendance records into meaningful business insights that help HR teams monitor workforce presence and make data-driven decisions.

---

# 🚀 Project Overview

Employee attendance data often contains valuable insights that remain hidden in spreadsheets. This project demonstrates how Power BI can be used to clean, model, and visualize attendance data, enabling HR professionals to track employee presence, identify trends, and improve workforce planning.

The dashboard provides an intuitive interface for exploring attendance metrics through interactive reports and visualizations.

---

# 🎯 Business Problem

HR teams need a centralized reporting solution to answer important business questions such as:

- What is the overall employee attendance percentage?
- How frequently are employees working from home?
- What is the trend in sick leave utilization?
- Which weekdays have the highest or lowest attendance?
- How does attendance change over weeks and months?
- Which employees have the highest leave utilization?

This dashboard addresses these questions through interactive KPIs and visual reports.

---

# 📂 Dataset Information

The project is built using an employee attendance dataset containing daily workforce records over multiple months.

### Dataset includes:

- Employee ID
- Employee Name
- Attendance Status
- Present Days
- Work From Home (WFH)
- Sick Leave (SL)
- Paid Leave (PL)
- Leave Without Pay (LWP)
- Weekly Off (WO)
- Holidays
- Other Leave Categories

The dataset was cleaned, transformed, and modeled in Power BI before creating the final dashboard.
